# Papers Directory

Place your publication PDFs here.

## Usage

1. Add your paper PDFs to this directory
2. Reference them in `src/data/publications.ts`:

```typescript
{
  title: "Your Paper Title",
  // ...
  paper: "/papers/yourpaper.pdf",
}
```

## Naming Conventions

Use descriptive names:
- ✅ `smith2024-neural-networks.pdf`
- ✅ `neurips2024-deep-learning.pdf`
- ❌ `paper1.pdf`
- ❌ `final_version_v3.pdf`

## Tips

- Keep file sizes reasonable (< 10MB)
- Use lowercase and hyphens
- Include year in filename for organization
- Consider hosting very large files externally (arXiv, conference site)
