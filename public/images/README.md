# Images Directory

Place your images here:

- `avatar.jpg` - Your profile photo (recommended size: 500x500px)
- `og-image.jpg` - Social media preview image (recommended size: 1200x630px)
- Project images (any format: jpg, png, webp)

## Quick Setup

1. Add your photo as `avatar.jpg`
2. Optionally add `og-image.jpg` for better social media sharing
3. Add project screenshots with descriptive names

## Image Optimization Tips

- Use WebP format for best performance
- Compress images before uploading
- Keep individual images under 500KB
- Use appropriate dimensions (don't upload massive images)

## Supported Formats

- JPG/JPEG
- PNG
- WebP
- SVG (for logos/icons)
- GIF (for animations)
