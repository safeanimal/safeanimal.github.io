Python爬虫-scrapy框架安装

发表于2021-04-29|更新于2022-04-15|编程Python爬虫
|阅读量:

Mac or Linux: pip install scrapy

Windows:

  1. pip install wheel

  2. 下载twisted:http://www.lfd.uci.edu/~gohlke/pythonlibs/#twisted

  3. 切换到twisted下载的目录后：pip install

  4. pip install pywin32

  5. pip install scrapy
