Python爬虫-scrapy使用middlewares设置headers和代理IP

发表于2021-04-30|更新于2022-04-15|编程Python爬虫
|阅读量:

目标：爬取豆瓣小说标签页面共计三页图书的信息。

流程：

  1. 在setting.py中根据需要设置LOG_LEVEL，ROBOTSTXT_OBEY，ITEM_PIPELINES，DOWNLOADER_MIDDLEWARES等项。
  2. 在middlewares.py中定义request.headers，用于UA伪装、IP代理等设置。
  3. 在items.py中定义数据项，用于封装spider_name.py解析到的数据。
  4. 在spider_name.py文件中定义URL和数据解析的逻辑（可能进行多层解析、不同URL的解析）。
  5. 在piplines.py中定义数据存储逻辑。
